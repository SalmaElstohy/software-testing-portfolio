## Bug ID
BR-001

## Title
Wrong success message after scanning QR code

## Environment
- App Type: Mobile Application
- OS: Android 13
- App Version: 1.2.0

## Preconditions
User is logged in

## Steps to Reproduce
1. Open the application
2. Scan QR code of a shipment
3. Observe the displayed message

## Actual Result
Success message appears: "Shipment scanned successfully"

## Expected Result
User should be redirected to Dispatch screen

## Severity
Medium

## Priority
High

## Status
Open
